# WPshop 1 vers Wpshop 2

Les entitées WPshop 1 et 2 sont similaires cependant l'entité **adresse** n'est plus existantes, en résumé nous retrouvons:

|          | WPshop 1 | WPshop 2 |
| Entitée  |          |          |
| Tier     | Oui      | Oui      |
| Contact  | Oui      | Oui      |
| Adresse  | Oui      | Non      |
| Produit  | Oui      | Oui      |
| Commande | Oui      | Oui      |
| Facture  | Non      | Oui      |

WPshop 2 proposes uniquement la migration des **tier**.
